# 📊 تعلم المحاسبة المالية

منصة تعليمية مجانية وشاملة لتعلم المحاسبة المالية من الصفر حتى الاحتراف.

🌐 **الموقع:** https://almubarmaj23-hub.github.io/financial-accounting-learn

## ✨ المميزات

- 🎓 **٨ وحدات تعليمية** تغطي كامل مناهج المحاسبة المالية
- 📖 **دروس تفاعلية** مع أمثلة عملية
- ✏️ **محرر القيود المحاسبية** مع التحقق التلقائي من التوازن
- 🧠 **اختبار تفاعلي** من ١٠ أسئلة متعددة الخيارات
- 📚 **قاموس المصطلحات** بالعربية والإنجليزية (٢٠+ مصطلح)
- 🌙 **الوضع الليلي** مدعوم بالكامل
- 📱 **تصميم متجاوب** لجميع الأجهزة
