# 📚 موقع تعلم المحاسبة المالية

موقع تعليمي متكامل لتعلم المحاسبة المالية باللغة العربية، مُصمَّم خصيصاً لطلاب الجامعات والمهتمين.

## 🌐 الموقع المباشر
> فعّل GitHub Pages من الإعدادات لنشر الموقع مجاناً

## 📄 صفحات الموقع

| الصفحة | الوصف |
|--------|-------|
| `index.html` | الصفحة الرئيسية |
| `pages/lessons.html` | 8 فصول تعليمية شاملة |
| `pages/quiz.html` | اختبار تفاعلي (10 أسئلة) |
| `pages/glossary.html` | قاموس 30+ مصطلح محاسبي |
| `pages/formulas.html` | المعادلات والنسب المالية |

## ✨ المميزات
- 🌙 وضع ليلي كامل
- 📱 تصميم متجاوب للجوال
- 🎯 اختبارات تفاعلية مع تصحيح فوري
- 📖 8 فصول تغطي المحاسبة من الأساسيات إلى القوائم المالية
- 🔍 بحث في المصطلحات
- ✅ شريط تقدم في الاختبار

## 🚀 تشغيل الموقع على GitHub Pages
1. اذهب إلى **Settings** في المستودع
2. اختر **Pages** من القائمة الجانبية
3. اختر **Branch: main** ثم احفظ

## 🛠️ التقنيات المستخدمة
- HTML5 | CSS3 | JavaScript
- Font Awesome Icons
- تصميم RTL كامل بالعربية
